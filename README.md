# Система автоматизации обработки обращений в службу техподдержки

## Описание проекта
Веб-система для автоматизации обработки обращений клиентов в службу техподдержки оптовой компании.

## Структура проекта
- `backend/` — серверная часть (Python/Django)
- `frontend/` — клиентская часть (HTML/CSS/JS)
- `database/` — скрипты БД, миграции
- `docs/` — документация, диаграммы
- `src/` — исходный код модулей
- `tests/` — тесты
